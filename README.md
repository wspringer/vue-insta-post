# README

A Vue component mimicking an Instagram post.

```javascript
import InstaPost from 'vue-insta-post'

Vue.use(InstaPost)
```

```html
<insta-post
  avatar="{img url}"
  handle="{user handle}"
  status="{status}"
  likes="{likes}"
  img-src="{img url}"
  @like="{like handler}"
/>
```
